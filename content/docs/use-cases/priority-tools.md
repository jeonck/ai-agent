## ROI / 난이도 매트릭스

```
임팩트
  ↑
높음│ B. 고임팩트    │ A. 우선 공략   │
   │ 고난이도 → 2단계│ 저난이도 → 1단계│
   ├────────────────┼────────────────┤
낮음│ D. 보류        │ C. 실험        │
   │ 고난이도        │ 저난이도        │
   └────────────────┴────────────────→
       어려움          쉬움          난이도
```

**A 영역 (우선 공략)**: 임팩트 높고 난이도 낮음 → 첫 PoC 후보
**B 영역**: 임팩트 높지만 어려움 → 역량 쌓은 후 도전
**C 영역**: 학습용·실험용
**D 영역**: 현 시점에선 보류

## 자동화 적합성 진단표

각 항목을 1-5점으로 평가 (합계 높을수록 에이전트 적합):

| 항목 | 평가 기준 | 점수 |
|------|----------|------|
| 반복성 | 매일/매주 반복되는 업무인가? | /5 |
| 규모 | 월 처리 건수가 100건 이상인가? | /5 |
| 구조화 가능성 | 입출력을 명확히 정의할 수 있는가? | /5 |
| 데이터 접근성 | 필요한 데이터에 API로 접근 가능한가? | /5 |
| 실패 허용도 | 오류 시 복구·검수 가능한가? | /5 |
| 전문성 불필요 | 고도의 전문 판단 없이도 처리 가능한가? | /5 |

**합계 해석**
- 25-30점: 에이전트 최적 후보
- 18-24점: PoC 가능, 리스크 관리 필요
- 12-17점: RAG 또는 워크플로우 재검토
- 12점 미만: 현재는 사람이 처리
